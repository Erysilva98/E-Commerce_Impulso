## Instalando o Projeto em sua Maquina

    - Clone o Repositório do Git em sua Maquina
    - Instalação das dependências do Projeto
        yarn install 

### Inicializando o Projeto

    - yarn start


### Em seu Navegador Acesse:
     http://localhost:3000
